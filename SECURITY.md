# 多平台 AI API 管理工具

一个安全、易用的多平台 AI API 密钥管理工具，支持硅基流动、阿里云 DashScope、DeepSeek、Kimi 等平台。

## 🔒 隐私与安全

**我们非常重视您的数据安全和隐私保护。**

### 完全本地存储
- ✅ **零服务器上传**：所有 API 密钥仅存储在您的浏览器本地 localStorage 中
- ✅ **无后端服务**：本应用为纯前端应用，不依赖任何后端服务器
- ✅ **无数据收集**：不收集、不上传、不存储任何用户数据

### AES-256 加密
- 🔐 所有密钥在本地使用 AES-256 加密算法加密存储
- 🔐 加密密钥随机生成，仅存储在您的浏览器中
- 🔐 即使有人访问您的 localStorage，也无法直接读取密钥

### 直连 API
- 🌐 直接向各平台官方 API 发送请求
- 🌐 不经过任何中间服务器或代理
- 🌐 API 请求直接从您的浏览器发出

### 您的数据，您做主
- 🗑️ 可随时清除浏览器数据来删除所有密钥
- 💾 支持导出数据到本地 JSON 文件
- 🔄 可在其他设备上导入备份数据

## 🛡️ 安全建议

1. **定期备份**：使用"导出数据"功能定期备份您的密钥
2. **安全的设备**：确保在安全的设备上使用本工具
3. **HTTPS 访问**：部署后确保通过 HTTPS 访问
4. **定期更换**：定期更换您的 API 密钥
5. **浏览器安全**：使用最新版本的现代浏览器
